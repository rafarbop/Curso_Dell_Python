Regressão Linear Múltipla¶

Neste notebook aprenderemos como utilizar a biblioteca scikit-learn para construirmos um modelo de previso de preo, usando regresso linear mltipla. Para isso utilizaremos um conjunto de dados de carros usados listados pelo site www.cardekho.com e disponvel em: https://www.kaggle.com/nehalbirla/vehicle-dataset-from-cardekho e adaptado neste notebook.

As informaes contidas no dataset so:

nome_carro: nome do carro

ano: ano de fabricao

preco_venda: preo que o dono quer vender

preco_atual: preo da concessionria

kms_rodados: kilometragem do carro

tipo_combustivel: tipo de combustvel do carro (Diesel, Gasolina, GasNatural)

tipo_vendedor: se o vendedor um revendedor ou a prpria pessoa

tipo_transmissao: tipo de cmbio do carro

n_donos: nmero de donos que o carro j teve

Separar dados em treino e teste¶

Para construir nosso modelo de Machine Learning precisamos dividir os dados em duas partes. Uma para treinar o modelo e outra para testar o modelo com dados que ele no conhece. Para isso iremos separar 80% dos dados para treino e 20% dos dados para teste. Para isso usaremos a biblioteca train_test_split que faz essa separao automaticamente.

In [ ]:

x_train, x_test, y_train, y_test = train_test_split(x, y, test_size=0.2)
